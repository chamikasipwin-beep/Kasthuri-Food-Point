# Kasthuri Food Point - Frontend

React frontend for Kasthuri Food Point restaurant website.

## Features

- Modern, responsive UI with dark mode
- Smooth animations using React Spring
- Category-based menu filtering
- Shopping cart with localStorage persistence
- Guest checkout (no login required)
- Order confirmation page

## Tech Stack

- React 18
- Vite
- React Spring (animations)
- Axios (API calls)
- Vanilla CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

The API base URL is configured in `src/services/api.js`. Update it for production deployment.
